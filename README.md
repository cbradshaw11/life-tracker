# Life Tracker

A personal life tracking app for monitoring health habits and other activities. Track smoking, workouts, drinking, and more with a calendar and year overview.

## Features

- **Calendar View** - Month grid showing which days you logged activities
- **Year View** - 12-month overview with activity density
- **Extensible Track Types** - Add custom activities (smoking, workout, drinking, meditation, etc.)
- **Local Storage** - Data persists in your browser (no backend required)
- **Export** - Backup your data as JSON

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS
- date-fns
- React Router
